Image Watermarking Desktop App

A simple Python desktop application that allows users to add text or image watermarks to their images. Users can preview the watermark in real-time, adjust opacity, position, and font size (for text), and save the final watermarked image.

Features

Upload and preview your main image

Add Text Watermark:

Customize font size

Adjust opacity

Choose position (Top-left, Top-right, Bottom-left, Bottom-right, Center)

Add Image Watermark:

Upload watermark/logo

Adjust opacity

Choose position

Live preview of the watermarked image

Save the final watermarked image in PNG or JPG format

Technologies Used

Python 3

Tkinter for GUI

Pillow (PIL) for image manipulation

Installation

Make sure you have Python 3 installed.

Install required libraries:

pip install pillow

Download or clone the repository.

Run the application:

python main.py

(Replace main.py with the filename of your Python script.)

Usage

Click Upload Image to select your main image.

Select Text Watermark or Image Watermark.

For text:

Enter your watermark text.

Adjust font size, opacity, and position.

For image:

Upload the watermark/logo image.

Adjust opacity and position.

Click Apply Watermark to preview the changes.

Click Save Image to export the final watermarked image.
